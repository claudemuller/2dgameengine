# Frostbyte❄️ Game Engine

A 2D game engine in C++ with SDL.

## Requirements

- [Lua 5.4](https://www.lua.org/)
- [SDL2.2](https://www.libsdl.org/)
- [gcc](https://gcc.gnu.org/)
- or [clang](https://clang.llvm.org/)
- [make](https://www.gnu.org/software/make/)

## Build & Run

```bash
make
```

### Build

```bash
make
```

### Run

```bash
make run
```

### Clean

```bash
make clean
```
